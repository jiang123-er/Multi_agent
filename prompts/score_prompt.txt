## 评分原则
1. 严格评分，平均分应控制在60-70分区间
2. 85分以上为优秀，仅给予真正突出的候选人
3. 应届生/无经验者，工作经验项上限12分
4. 发现问题必须扣分，不要因为是应届生就放水
5. 教育背景直接查表，知名985硕士=16分，不要给14分

## 评分标准（总分100分）

### 1. 教育背景（20分）

**院校档次判断规则**：
- **顶尖院校**：清华、北大、复旦、上交、浙大、中科大、南大、西交、哈工大（C9联盟）
- **知名985**：北航、北理、同济、华科、武大、中山、厦大、天大、南开、东南、电子科大等
- **普通985/211**：其他985或211院校
- **海外名校**：QS/Times/USNews排名前100

| 分数 | 标准 |
|------|------|
| 20分 | C9博士 |
| 18分 | C9硕士 或 知名985博士 |
| 16分 | 知名985硕士 或 普通985/211博士 |
| 14分 | 知名985本科 或 普通985/211硕士 |
| 12分 | 普通985/211本科 或 普通一本硕士 |
| 10分 | 普通一本本科 |
| 7分 | 二本本科 |
| 4分 | 民办本科 |
| 0分 | 大专或无教育背景 |

**计算步骤**：
1. 判断院校档次
2. 判断学历层次
3. 查表得出分数

**院校识别提示**：
- "XX学院"结尾（如武汉XX学院、广州XX学院）→ 通常是民办本科 → 4分
- "XX大学"结尾（如武汉理工大学、广州大学）→ 公办本科 → 根据档次评分
- 校名带"职业"、"民办"字样 → 民办本科 → 4分

### 2. 技能匹配度（25分）
| 分数 | 标准 |
|------|------|
| 25分 | 技能完全匹配，有多个深度项目佐证，能独立负责模块 |
| 20分 | 核心技能匹配，有1-2个项目经验，需少量指导 |
| 15分 | 技能基本匹配，项目经验较浅，需要较多培训 |
| 10分 | 部分技能匹配，缺乏项目实战经验 |
| 5分 | 技能严重不匹配，需要从头培训 |
| 0分 | 无技能信息 |

**扣分项**：
- 技能描述模糊（如"熟悉XX"但无项目佐证）：-3分
- 技能数量过多但缺乏深度：-5分

### 3. 经验与潜力（25分）

**有工作经验者**：
| 分数 | 标准 |
|------|------|
| 25分 | 同行业同岗位3年以上，有明确量化业绩 |
| 20分 | 相关岗位2年以上，有较好业绩 |
| 16分 | 相关岗位1-2年，业绩描述一般 |
| 12分 | 经验较少或相关性弱 |
| 8分 | 无相关经验但有其他工作经验 |

**应届生/无经验者**（上限12分）：
| 分数 | 标准 |
|------|------|
| 12分 | 有高质量实习经历（大厂/知名公司），表现优秀 |
| 10分 | 有相关实习经历，表现良好 |
| 6分 | 无实习，但有较好的项目实践经历 |
| 4分 | 无实习，项目经历较少 |
| 0分 | 无任何实践经历 |

**大厂/知名公司判断**：
- **互联网大厂**：字节跳动、阿里、腾讯、百度、美团、京东、拼多多、快手、小红书、网易、滴滴、华为、小米
- **AI公司**：OpenAI、Anthropic、商汤、旷视、科大讯飞、智谱AI、月之暗面
- **金融/券商**：中金、中信、华泰、国泰君安、头部银行总行
- **外企**：Google、Microsoft、Amazon、Apple、Meta、特斯拉

**判断提示**：
- 看到"字节跳动"实习 → 大厂 → 15分
- 看到"阿里/腾讯/百度"实习 → 大厂 → 15分
- 看到"小公司/不知名公司"实习 → 12分
- 看到"无实习" → 8分或更低

**扣分项**：
- 空窗期超过6个月：-5分
- 频繁跳槽（1年内换2家以上）：-5分

### 4. 项目质量（20分）
| 分数 | 标准 |
|------|------|
| 20分 | 项目复杂度高，技术栈先进，个人贡献明确，有量化成果 |
| 16分 | 项目有一定复杂度，技术栈主流，个人贡献清晰 |
| 12分 | 项目中等复杂度，描述较完整 |
| 8分 | 项目简单（CRUD级别），或描述模糊 |
| 5分 | 项目为课程作业级别，或仅有简单功能 |
| 0分 | 无项目经历 |

**扣分项**：
- 项目描述模板化、无个人特色：-3分
- 个人贡献占比描述不清：-2分

### 5. 整体印象（10分）
| 分数 | 标准 |
|------|------|
| 10分 | 结构清晰，重点突出，无任何问题 |
| 8分 | 结构规范，信息完整 |
| 6分 | 结构一般，信息基本完整 |
| 4分 | 结构混乱，或关键信息缺失 |
| 2分 | 严重混乱，难以阅读 |
